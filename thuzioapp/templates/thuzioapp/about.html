This app is a proof of concept/MVP to learn the django framework. Future implementations for a production quality application must require the following...
  • Utilize forms / formfactory
    • Dynamically generate form objects to pass into templates.
    • Add validation on forms
  • Examine utilization of cache and user cases that could cause issues
  • Explore further abstraction of database models to limit excessive fields (perhaps store internal marketing related info into a non-relational db)
  • Add image processor to app for product images
  • Store/read images from s3 bucket for product images
  • Add redirect history
  • Add account options (reset/forgot password, remember me, etc)
  • Refactor front end JS (in particular the detail template (indexing elements is not scaleable)
  • Implement Promo's
    • Add promo boolean field onto product in db
    • Add promo img src onto product in db = stored on s3
    • Add query on index action
    • Pass that product promo img into index template through request context and display the promo
    • Add functionality into adminCP so staff can choose which product to promo
    • Create image carousel to support multiple promo's
    • Add validation to ensure set # of promo's in carousel
  • Unit testing
  • Responsive design for mobile web users
  • Multibrowser compatibility testing
  • Cronjob / rake task to clean up and/or track incomplete purchases and schedule reminder emails
  • Refactor/optimize views
  • Remove unnecesssary fields from db table
  • Add categories to product table and implement search by categories
